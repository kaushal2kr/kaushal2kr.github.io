Learning Roadmap

This is the order I am following to build strong technical fundamentals.

Step 1: Networking Foundation

Learn OSI model, TCP/IP model, IP addressing, subnetting, ARP, DNS, DHCP, switching, routing, VLANs, and basic troubleshooting.

Step 2: Packet Analysis

Use Wireshark to understand Ethernet frames, IP packets, TCP/UDP, DNS queries, ARP packets, HTTP/HTTPS traffic, and real packet flow.

Step 3: Cybersecurity & SOC Basics

Study logs, SIEM alerts, phishing, malware basics, incident response, false positives, IOC, MITRE ATT&CK, and SOC L1 investigation flow.

Step 4: Practical Labs & Projects

Build small visual labs and write-ups that explain real technical concepts in a simple and interview-friendly way.
